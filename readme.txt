== Description ==
You get a complete list of the products purchased by a current user. If the same product is purchased more than once by the same user then it doesn&apos;t show in the list every time.

You may use shortcode [boughtproduct] for get list of product.

== Installation ==
1. Upload `woo-bought-products` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. That\'s it you are ready to go :) .
